coordinadora empresa transportadora de Colombia

== Description ==

Integración de Coordinadora como método de envio para woocommerce podrá usar los servicios: generacion de guias,
seguimiento de despachos. Actualmente en esta versión solo permite calcular el costo de envio.

* Nota:! Instale este versión y solicite el servicio de Coordinadora mercantil.

== Installation ==

1. Descargar el plugin
2. Ingrese a su administrador de wordpress
3. Ingrese al menu plugins / añadir nuevo / subir plugin.
4. Encuentra el plugin descargado en tu computadora y cárgalo como cualquier otro archivo.
5. Después de instalar el .zip, puede verlo en la lista de complementos instalados, puede activarlo o desactivarlo.
6. Active el plugin y siga el proceso de configuración.
7. ** Asegúrese por favor de establecer tanto el país, estado y ciudad donde se ecnuentra ubicada la tienda
